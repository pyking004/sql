Data Merging:
=============
It is nothing but merging more than one table data into a single set.

There are two types of data merging 

1. Vertical data merging ==> set operators
2. Horizontal Data merging ==> Joins

Vertical Data Merging:
======================
Merging more than one table data with set operators known as vertical merging

Here one table data below another table data is available known as column wise merging

SET OPERATORS:
==============
UNION,	UNION ALL, INTERSECT, MINUS

syntax:
=======
select statement1
set operator
select statement2;

Rules to work with set operators:
===========================

Rule1:
=====
No of columns in first select statement should be same with no of columns in second select statement.

Rule2:
======
Datatypes of first select statement columns should be similar with datatypes of second select statement columns.
