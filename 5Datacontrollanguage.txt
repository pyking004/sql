DCL: DATA CONTROL LANGUAGE:
===========================
It is used to control the data from one user to another user.

GRANT  ,  REVOKE

alter user system identified by Oracle1234 account unlock;

Grant: 
======
It is used to provide privileges from one user to another user

syntax:
=======
	grant <privileges> on <object> to <users>;

revoke: It is used to cancel  already given privileges.
======

syntax:
=======
	revoke <privileges> on <object> from <users>;


Note:
====
*to practice DCL we required more than one user

to see all the users in database
================================
sql>select * from all_users;

How to create new users
=======================
step1: login with admin account as below

user name : system
password  : Oracle1234 ( At the time of installation provided password)

sql>show user
user is "system"

step2: create new user with below statement

sql>create user user1 identified by tiger;
sql>grant connect,resource,unlimited tablespace to user1;

sql>create user user2 identified by tiger;
sql>grant connect,resource,unlimited tablespace to user2;
